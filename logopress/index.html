Soluciones integradas

Logopress™ DieDesign

Diseño completo del troquel dentro de SOLIDWORKS®.

  • • Desarrollo/optimización de tira.
  • • Inserción automática del Die Set.
  • • Librerías de componentes (DME, Misumi, etc.).
  • • Planos y listas de materiales automáticas.

Logopress™ ProgSim

Simule el formado para anticipar arrugas, fracturas o adelgazamiento.

  • • Análisis de formabilidad.
  • • Menos pruebas y modificaciones.
  • • Calidad consistente desde el inicio.

Herramientas específicas

Logopress™ Flatten

Obtenga el blank preciso para piezas complejas con un clic.

  • Cotizaciones rápidas y precisas.
  • Mejor aprovechamiento de material.
  • Punto de partida para la tira.

Logopress™ Blank

Defina el perfil de chapa para embutidos de paredes rectas.

  • Acelera el diseño de embutido profundo.
  • Aumenta la precisión desde el inicio.
